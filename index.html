Single Character Variables only

How to use it?
  1. You can declare single charater Variables by simply typing in the text field above for example: x=10, but doing so will erase this "How To" so read before you continue.
  2. While using the calculator, please keep in mind that using parenthesis should be followed by your desired operator for example cos(3)*100, without the operator after closing paranthesis cos(3)100 will not work.
  3. Calculation history will be saved which you can remove one by one at any time, and by clicking on the highlighted result of the history expressions you can use that value in a new expression.
